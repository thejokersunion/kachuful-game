## Key Interactions Summary

### Tap Actions

1. **Card Tap** (Your Turn):
   - First tap: Select card (highlight)
   - Second tap: Deselect OR auto-play if only one playable

2. **Bid +/- Buttons**:
   - Tap to increment/decrement
   - Shows immediate visual feedback

3. **Confirm Button**:
   - Glows when ready
   - Disabled (greyed) when not your turn

4. **Copy Buttons**:
   - Changes icon to checkmark on copy
   - Shows "Copied!" toast message

### Visual Feedback

1. **Connection Status**:
   - Green dot: Connected
   - Red dot: Disconnected
   - Yellow dot: Reconnecting

2. **Turn Indicator**:
   - Pulsing border on current player
   - "Your Turn!" text
   - Optional sound/vibration

3. **Card Legality**:
   - Bright: Can play
   - Dim: Cannot play
   - Border color: Green (trump), Red (led suit)

## Responsive Adaptations

### Mobile Portrait (< 768px)

- Single column layout
- Stacked player areas
- Bottom sheet for scores
- Larger touch targets (60px)
- Simplified animations

### Tablet Landscape (768px - 1024px)

- Two-column layout
- Side-by-side opponent views
- More spacing between elements
- Medium touch targets (52px)
- Full animations

### Desktop (> 1024px)

- Full table view (circular arrangement)
- All players visible simultaneously
- Hover effects enabled
- Mouse-optimized controls
- Enhanced animations
